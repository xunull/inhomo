# 节点地区只按节点名解析，不做权威 GeoIP

mihomo 的 `/connections` 与 `/proxies` API 都不暴露代理节点的服务器 IP（出于隐私），因此无法用「节点 IP → GeoIP」得到权威的节点所在国家。我们决定：出境节点的地区标签一律从节点**名字**（国旗 emoji / 国家字样）尽力解析，解析不出就记 `unknown`；地区仅作分类/排序标签，**绝不**作为告警的硬性筛选条件——告警条件由「明文 HTTP + 经出境节点中转」决定，与地区无关。

代价：名字不含地区信息的节点（如 `Node-3`、`中转-A`）会是 `unknown`。好处：零额外依赖、不受 API 限制。若未来 mihomo 暴露节点服务器地址，可重开此决策。
